# buoyant_entrainment
Code used in "Buoyancy-Driven Entrainment in Dry Thermals" by McKim et al. (2019) to analyze simulations of dry convection.

For ease of understanding, the files best read in the following order:
- h5_to_netcdf.py
- tracking.py
- vorticity.py
- cross_section.py
- domega_dt.py
- circulation.py
- momentum.py
- scalings.py
- entrainment.py
- g_snapshot.py
- efficiency.py
